## Run Locally

Clone the project

```bash
  git clone https://github.com/SYShopnil/ATPL-Task.git
```

Go to the project directory

```bash
  cd
  ATPL-Task
```

Install dependencies

```bash
  npm install || npm i
```

Start the server

```bash
  npm run dev
```

Start the Test

```bash
  npm run test
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/SYShopnil/m360ict-book-library-task.git
```

Go to the project directory

```bash
  cd
  m360ict-book-library-task
```

```bash
  cd
  view
```

Install dependencies

```bash
  npm install || npm i
```

Start the server

```bash
  npm run dev
```

Start the Test

```bash
  npm run test
```
